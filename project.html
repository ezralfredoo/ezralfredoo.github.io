Java Application Project: EzSan Rupiah Currency Converter

Ini merupakan project mata kuliah Pemrograman dan Algoritma sewaktu saya semester 1. Pada waktu itu diadakan pameran yang berisi 18 aplikasi buatan mahasiswa IMT angkatan 2019. Saya bersama teman saya yang bernama Nicky membuat aplikasi java yang berfungsi sebagai converter mata uang berdasarkan Rupiah. Pada aplikasi ini tersedia pilihan 8 mata uang dari berbagai negara yang dapat dikonversikan menjadi nilai rupiah atau sebaliknya. Aplikasi ini kami namakan EzSAn Rupiah Currency Converter.
